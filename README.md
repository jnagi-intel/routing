# Routing

To compile both ssp and routing 
	make all
To compile only ssp
	make ssp
To compile only routing 
	make routing 
Delete all executables and object files 
	make clean 

To run ssp
	./ssp input_1000000.txt 0 999999
To run routing 
	./routing input_graphsmall_part2.txt input_ipsmall_part2.txt 0 99
